# Задачка для интервью

Базовая задачка на ревью и лайв-кодинг.

Тут приведен очень плохой код веб-краулера, который принимает на вход список ссылок,
нужно его отревьюить и исправить.

## Подготовка окружения

Для запуска нужен uv, устанавливается [тут](https://docs.astral.sh/uv/reference/installer/#changing-the-installation-path).

Для установки зависимостей нужно запустить

```bash
uv run sync --all-groups
```

## Запуск приложения

```
uv run -m crawler -i input_data.txt -o out
```
